# Project Skills

- **Resume Parsing**: PDF and DOCX extraction using `pypdf` and `python-docx`.
- **Vector Store**: ChromaDB for storing and retrieving embedded resume chunks.
- **Embeddings & LLMs**: Integration with OpenAI, Google Gemini, Anthropic, Mistral, DeepSeek via OpenRouter.
- **Web UI**: Streamlit frontend with custom CSS for a modern dark‑theme experience.
- **Hybrid Search**: Combination of keyword BM25 and semantic similarity for accurate results.
- **Environment Management**: `.env` handling with `python-dotenv` for API keys.
- **Model Selection**: Dynamic dropdown to switch between multiple LLM providers.
- **Real‑time Indexing**: On‑demand re‑indexing of resumes folder.
- **Source Attribution**: Displays source documents for each answer.

These skills enable the app to act as a resume‑intelligence assistant, answering candidate queries directly from indexed resume content.
